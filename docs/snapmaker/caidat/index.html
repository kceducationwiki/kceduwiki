Cài đặt ứng dụng Snapmaker luban cho máy tính

Yêu cầu về Hệ thống và Phiên bản​

Trước khi tải xuống và cài đặt, hãy đảm bảo rằng máy tính của bạn đáp ứng các yêu cầu hệ thống tối thiểu và xác định phiên bản cài đặt phù hợp để đảm bảo khả năng tương thích với sản phẩm Snapmaker của bạn.

Tải Snapmaker Luban​

Snapmaker Luban là phần mềm chính thức giúp bạn:

  • Kết nối và điều khiển máy in 3D Snapmaker
  • Tối ưu hóa quy trình thiết kế và in ấn
  • Đảm bảo hiệu suất hoạt động cao và độ chính xác

Nguồn tải phần mềm:​

  • Trang chính thức Snapmaker
  • Snapmaker Luban trên GitHub

Chọn đúng phiên bản theo loại chip:​

  • 💻 Mac với chip M1/M2: Tải file arm64.dmg
  • 💻 Mac với chip Intel: Tải file x64.dmg
  • 💻 Windows: Tải file .exe

Cài đặt Snapmaker Luban​

Trên Windows​

  1. Nhấp đúp vào tệp .exe để chạy trình cài đặt
  2. Nhấn Browse để chọn thư mục cài đặt
  3. Nhấn Install để bắt đầu quá trình
  4. Khi hoàn tất, nhấn Finish

Thiết lập ban đầu​

Sau khi cài đặt, khởi chạy Snapmaker Luban. Làm theo Configuration Wizard để:

  • Chọn ngôn ngữ giao diện
  • Chọn loại máy Snapmaker bạn đang sử dụng
  • Chọn đầu công cụ (Toolhead)
  • Chọn mô-đun tương ứng

Cập nhật Snapmaker Luban​

1. Cập nhật tự động​

  1. Khi có phiên bản mới, cửa sổ "Update Snapmaker Luban" sẽ tự động hiển thị.
  2. Nhấn Update Now để cập nhật.

Nếu bạn không muốn Luban tự động hiển thị thông báo cập nhật, hãy làm theo:

Settings > Preferences > General > Software Update

Bỏ chọn Tự động kiểm tra cập nhật.

2. Cập nhật thủ công​

  1. Vào Settings > Preferences
  2. Chuyển đến General > Software Update Nhấn Check for updates
  3. Nếu có bản mới, nhấn Update Now

⚠️ Nếu xảy ra sự cố trong quá trình cập nhật, hãy gửi phiếu hỗ trợ đến đội ngũ kỹ thuật Snapmaker.

Tải xuống và Cập nhật Thủ công​

Bạn có thể tải thủ công phiên bản mới nhất hoặc phiên bản cụ thể từ:

  • Snapmaker Website
  • Snapmaker Luban GitHub Releases

Liên hệ hỗ trợ​

Nếu gặp khó khăn trong quá trình cài đặt hoặc cập nhật, hãy liên hệ với bộ phận hỗ trợ của Snapmaker qua trang hỗ trợ chính thức.
